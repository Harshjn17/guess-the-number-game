# 🎯 Guess the Number Game

A simple JavaScript game where the player has to guess a randomly generated number between **1 and 100**.

## 🚀 Features

- Random number generated every game
- Hint system
  - 📈 Too High
  - 📉 Too Low
  - 🎉 Correct Guess
- Guess counter
- Input validation (1–100)
- Game ends after the correct guess
- Responsive UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 📂 Project Structure

```
Guess-Number-Game/
│── index.html
│── style.css
│── script.js
│── README.md
```

## ▶️ How to Run

1. Download or clone the repository.
2. Open `index.html` in your browser.
3. Enter a number between **1 and 100**.
4. Click the **Guess** button.
5. Keep guessing until you find the correct number.
